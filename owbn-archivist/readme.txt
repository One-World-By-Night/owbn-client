OAT (OWBN Archivist Toolkit) — workflow engine, character registry, submissions, inbox, reports, and custom content hub.

== Description ==

Provides the Archivist Toolkit (OAT) module for OWBN sites, including:

* Inbox — assignments and watched entries
* Submit — create new workflow entries
* Registry — character registry with detail views
* Reports — aggregated data for local OAT hosts
* Custom Content Hub — browse and manage curated content
* Gateway endpoints for remote OAT data access
* Elementor widgets for frontend display

Requires the OWBN Core plugin.
